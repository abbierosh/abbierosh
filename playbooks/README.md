# Playbooks

This folder holds reusable investigation queries.

These are the SQL checks I reach for when I need to answer basic but important questions quickly:

- does the record exist?
- did an expected interaction happen?
- when did activity start and stop?
- is the issue isolated or widespread?

The goal is not to produce a perfect final report. The goal is to get to a reliable operational answer fast enough to guide the next step.

Each playbook is written with comments because the thinking matters as much as the query itself.
